# TaskMaster - Django CRUD Application

A modern, premium CRUD (Create, Read, Update, Delete) application built with Django 6.0 and curated with a sleek glassmorphic UI.

## 🚀 Features
- **Full CRUD Functionality**: Manage tasks seamlessly.
- **Premium UI**: Dark-themed, responsive design using Vanilla CSS.
- **Form Validation**: robust data handling with Django Forms.
- **Clean Architecture**: Organized into separate apps and templates.

## 🛠️ Technology Stack
- **Backend**: Django 6.0 (Python)
- **Frontend**: HTML5, Vanilla CSS (Outfit Google Font)
- **Database**: SQLite

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Janak27-del/crudwithdjango.git
   cd crudwithdjango
   ```

2. **Run Migrations**:
   ```bash
   python manage.py migrate
   ```

3. **Start the Development Server**:
   ```bash
   python manage.py runserver
   ```

4. **Access the App**:
   Open [http://127.0.0.1:8000/](http://127.0.0.1:8000/) in your browser.
